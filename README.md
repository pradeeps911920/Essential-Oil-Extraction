# Essential Oil Extraction from Lemongrass using Steam Distillation

## Overview
This project was carried out as a Chemical Engineering course project at IIT Gandhinagar. The objective was to extract essential oil from fresh lemongrass using a laboratory-scale steam distillation setup and analyze the extracted oil using FTIR spectroscopy.

---

## Objectives
- Extract essential oil from fresh lemongrass using steam distillation.
- Study the effect of operating parameters on extraction efficiency.
- Characterize the extracted oil using FTIR spectroscopy.

---

## Methodology
1. Feed preparation of fresh lemongrass.
2. Steam distillation using a laboratory-scale setup.
3. Collection and separation of essential oil.
4. Experimental data analysis.
5. FTIR characterization of the extracted oil.

---

## Results
- Successfully extracted essential oil from lemongrass.
- Confirmed the presence of major bioactive compounds such as citral and terpenes through FTIR analysis.
- Studied extraction efficiency and process performance under different operating conditions.
